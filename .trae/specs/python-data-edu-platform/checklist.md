# Python 商务数据分析在线教育平台 - 验证清单

- [ ] Checkpoint 1 (本地预览): 在仓库根目录执行 `python -m http.server 8000`，浏览器访问 `http://localhost:8000` 出现首页，无白屏、无 404
- [ ] Checkpoint 2 (首页内容): 首页包含"平台介绍 / 课程列表入口 / 学习中心入口 / 成就中心入口"四个可见区块
- [ ] Checkpoint 3 (课程数量): 进入课程列表页可看到 10 门课程卡片，每张显示标题、简介、技能标签、难度、预计时长
- [ ] Checkpoint 4 (课程路由): 点击任意课程卡进入课程详情页（URL 形如 `/course.html?id=xxx` 或 hash 路由），课程标题与简介正确
- [ ] Checkpoint 5 (章节切换): 课程详情页左侧或顶部有章节 TOC；点击章节 N 切换内容；可按"上一章 / 下一章"按钮顺序导航
- [ ] Checkpoint 6 (代码复制): 每个代码块右上方有"复制"按钮，点击后 1 秒内提示"已复制"，剪贴板文本与代码块一致
- [ ] Checkpoint 7 (代码下载): 每个代码块有"下载 .py"按钮，点击触发浏览器下载，文件名为课程/章节相关命名且内容为纯 Python 文本
- [ ] Checkpoint 8 (章节练习): 每章至少 3 道题，支持单选/多选/填空/代码填空；提交后每题显示对错与解析，章节得分显示
- [ ] Checkpoint 9 (章节判分精度): 全对 → 100；全错 → 0；3/5 对 → 60（比例）；刷新后章节得分仍然存在
- [ ] Checkpoint 10 (综合测评): 每门课程有"综合测评"入口，≥ 10 题；提交后显示总分与是否通过（≥ 60 为通过）
- [ ] Checkpoint 11 (进度持久化): 关闭浏览器后重新打开，首页仍显示课程完成度、徽章与总积分；提供"导出学习记录"与"重置数据"按钮
- [ ] Checkpoint 12 (重置功能): 点击"重置数据"→ LocalStorage 中学习记录 key 被清除；首页数据恢复初始态
- [ ] Checkpoint 13 (成就/徽章):
  - 完成第 1 门课程 → "第一门课程"徽章点亮并弹出 toast
  - 连续 7 天访问（可通过模拟日期测试）→ "坚持 7 天"徽章点亮
  - 某测评满分 → "满分学员"徽章点亮
- [ ] Checkpoint 14 (成就中心): 访问 `/achievements.html` 可看到已获徽章、未获徽章、总积分与排名（含虚拟同学对比）
- [ ] Checkpoint 15 (学习仪表盘): 首页可见"已完成课程数、总积分、已获徽章数、今日推荐、继续学习入口"
- [ ] Checkpoint 16 (Python 资源下载): 每门课程详情页有"下载完整代码与数据"按钮，点击下载 zip 或多个 `.py/.csv`
- [ ] Checkpoint 17 (本地运行课程代码): 任选第 3 门（RFM）与第 8 门（时间序列），使用课程提供的 Python 代码与数据，在本地 Jupyter 中可成功运行并生成图表
- [ ] Checkpoint 18 (搜索/筛选): 课程列表页支持搜索关键词（如"RFM"）与标签多选筛选；结果即时更新
- [ ] Checkpoint 19 (响应式 / 移动端): 开发者工具切换到 375×812 / 768×1024 / 1440×900 均无横向滚动条；课程页、练习页、仪表盘布局正常
- [ ] Checkpoint 20 (可部署性):
  - 将仓库连接 Cloudflare Pages，构建命令留空，输出目录 `/`
  - Pages 构建日志显示"Success"，访问 `*.pages.dev` 域名：首页、课程页、练习页均正常加载；无白屏
- [ ] Checkpoint 21 (性能): 首屏 HTML/CSS/JS 合计 ≤ 500KB；LCP ≤ 2.5s（Chrome DevTools Lighthouse 或实际观察）
- [ ] Checkpoint 22 (浏览器兼容): Chrome / Edge / Firefox / Safari 最新两个主版本均能正常做题与复制
